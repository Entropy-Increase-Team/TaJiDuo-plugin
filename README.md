# TaJiDuo-plugin

基于 TaJiDuo API 的 Yunzai-Bot **塔吉多 / 幻塔 / 异环**插件

[安装](#安装插件) · [配置](#配置说明) · [功能](#当前功能) · [数据存储](#数据存储)

</div>

---

- 一个适用于 Yunzai 系列机器人框架的塔吉多账号与游戏数据查询插件
- 当前版本：`0.1.0`
- 支持网页登录、多账号管理、幻塔/异环角色查询、签到、社区任务、商城与兑换码查询
- 默认后端：`https://tajiduo.shallow.ink`

## 使用须知

本插件依赖 TaJiDuo 后端接口。使用前请先在 `config/common.yaml` 中填写可用的 `api_key`。

账号登录推荐使用网页登录：发送 `塔吉多登录` 或 `tjd登录`，机器人会返回一个在线登录链接。登录完成后，当前账号会自动设为默认账号。

## 安装插件

在 Yunzai 根目录执行：

```bash
git clone https://github.com/Entropy-Increase-Team/TaJiDuo-plugin.git ./plugins/TaJiDuo-plugin
```

### 安装依赖

插件直接依赖如下：

| 依赖 | 用途 |
|------|------|
| `express` | 网页登录本地 HTTP 服务 |
| `yaml` | 读取与解析 YAML 配置 |
| `chokidar` | 监听配置文件变更 |

如果你的 Yunzai 环境已经安装这些依赖，可以直接重启机器人；否则进入插件目录安装：

```bash
cd ./plugins/TaJiDuo-plugin
pnpm install
```

没有使用 pnpm 的环境可改用 `npm install`。

目录结构大致如下：

```text
plugins/TaJiDuo-plugin
├── apps/          # 命令入口
├── components/    # 网页登录服务
├── config/        # 用户配置
├── defSet/        # 默认配置
├── model/         # API 与账号模型
├── resources/     # 帮助图与网页登录页面
└── utils/         # 通用工具
```

## 配置说明

### common.yaml

```yaml
# TaJiDuo API 服务地址
base_url: "https://tajiduo.shallow.ink"
# API Key，通过 X-API-Key 请求头传递
api_key: ""
# 请求超时时间，单位毫秒
timeout: 25000

# 网页登录服务配置，发「tjd登录」时会生成此服务下的登录链接
login_server:
  enabled: true
  port: 25188
  public_link: "http://127.0.0.1:25188"

# 社区任务轮询设置
community_task:
  action_delay_ms: 3000
  step_delay_ms: 8000
  poll_times: 8
  poll_interval_ms: 5000
```

`public_link` 是发给用户打开的登录地址。如果机器人部署在服务器或反代后面，请改成公网可访问地址，例如 `https://tjd.example.com`。

`X-Platform-Id` 会直接使用当前机器人账号 ID，不再提供单独的 `platform_id` 配置项。

### sign.yaml

```yaml
auto_sign: false
auto_sign_cron: "10 1 * * *"
games:
  - huanta
  - yihuan
notify_list:
  friend: []
  group: []
```

开启 `auto_sign` 后会按 cron 自动执行配置游戏的签到，并向 `notify_list` 中配置的好友或群发送结果。

## 当前功能

命令前缀支持中文与缩写形式：

| 类型 | 前缀 |
|------|------|
| 塔吉多 | `塔吉多` / `tjd` / `#tjd` / `/tjd` |
| 幻塔 | `幻塔` / `ht` / `#ht` / `/ht` |
| 异环 | `异环` / `yh` / `#yh` / `/yh` |

### 插件基本

| 命令 | 说明 |
|------|------|
| `塔吉多帮助` / `tjd帮助` | 打开帮助菜单 |

### 塔吉多账号

| 命令 | 说明 |
|------|------|
| `塔吉多登录` / `tjd登录` | 获取网页登录链接 |
| `塔吉多登录列表` | 查看本地已保存账号 |
| `塔吉多切换登录 <序号>` | 切换当前默认账号 |
| `塔吉多删除登录 <序号>` | 删除指定账号 |

## 数据存储

- 账号数据保存到 Redis：`TJD:USER:${QQ}`
- 短信验证码临时数据保存到 Redis：`TJD:CAPTCHA:${QQ}`，有效期 10 分钟
- 网页登录会话保存在插件内存中，登录完成或超时后会清理
- 本地账号数据会保存 `fwt`、昵称、UID、平台账号 ID、绑定时间等字段，请不要公开 Redis 数据
